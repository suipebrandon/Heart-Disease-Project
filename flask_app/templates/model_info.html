Model Information

Academic summary of the dataset, preprocessing, model, and limitations.

Dataset

The project uses the UCI Heart Disease dataset file included in this repository. The local CSV contains 920 patient records from Cleveland, Hungary, Switzerland, and VA Long Beach.

Model Pipeline

  • Missing numeric values were handled during notebook preprocessing.
  • Categorical features were one-hot encoded.
  • Feature scaling was applied using StandardScaler.
  • Several classifiers were compared, including Logistic Regression, SVM, Decision Tree, and Random Forest.
  • Random Forest was selected and tuned using GridSearchCV.

Reported Performance

The project notebook and README report approximately 85% accuracy and about 0.90 ROC-AUC for the selected model. These metrics should be discussed as experimental results from the dataset, not as clinical validation.

Accuracy
~85%

Limitations

  • The application is for academic and educational use only.
  • The model was trained on a public dataset and has not been clinically validated.
  • Some source dataset fields contain missing values, which can affect model reliability.
  • The prediction is decision-support information, not a medical diagnosis.
  • A healthcare professional should interpret any real patient case.
